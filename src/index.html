WELCOME TO THE CITY OF DETROIT PROPERTY SEARCH MAP

Navigate the map to search for property information across Detroit. Key features to explore:

  • Look up a specific address in the Find Property search bar
  • Select search criteria with Data Layers to explore opportunities
  • Click on a parcel to display property information
  • Apply to purchase publicly owned property by clicking on the “Inquire to buy this Property” button in the parcel information feature
  • Click the Info icon to see more detail about the property layers
